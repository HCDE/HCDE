# HCDE

HCDE is a multiplayer-focused DOOM source port project built for modern co-op and competitive play.
It focuses on netcode reliability, mod compatibility, and practical server tooling.

# 2026 Project Section

HCDE is actively developed by B0k0 and the HCDE Team.
Current priorities include:

- authority handoff and sync stability
- launcher and updater quality-of-life improvements
- compatibility guards for community mods
- dedicated server reliability for LAN and public rooms

# Acknowledgments

Huge thanks to the DOOM open-source ecosystem and the modding community.
HCDE exists because of years of shared engine work, testing, and community content.
